Uma plataforma desenvolvida para o [Sebrae](https://sebrae.com.br/sites/PortalSebrae/). Foi desenvolvida com o intuito de centralizar varias empresas apuradas pelo [Sebrae](https://sebrae.com.br/sites/PortalSebrae/) para que os usuários possam acessar e visualizar as informações de cada empresa.

A plataforma oferece um layout moderno totalmente pensado na experiencia de usuário.

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Todo o processo de desenvolvimento teve um prazo de um mês, contando com apenas um desenvolvedor front-end, o que significa que trabalhei com apoio da equipe de dados e design.

O desenvolvimento foi concluído antes do prazo, aonde foi feito o desenvolvimento da plataforma, configuração do servidor e deploy. Alem de outras configurações para a plataforma suportar mais de mil acessos simultâneos.

## Entrega do projeto

Foi entregue junto com o projeto uma funcionalidade que sugeri. Armazenar logs específicos para saber quais foram os cards mais acessados durantes as três datas do evento.

Por final, a plataforma foi entregue e apresentada no [Startup Summit 2024](https://www.startupsummit.com.br/) e contou com mais de mil acessos simultâneos.

Até o momento conta com atualizações e segue sendo apresentada em eventos.
