Uma aplicação completa desenvolvida como **assessment técnico** para a Prefeitura Municipal de Florianópolis (PMF), com o objetivo de avaliar diferentes cenários de uso, arquitetura e qualidade de código em um contexto real de sistema público.

## Tecnologias utilizadas

- [Next.js](https://nextjs.org/) (App Router + Server Components)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Query](https://tanstack.com/query/latest)
- [Shadcn/ui](https://ui.shadcn.com/)
- [Zod](https://zod.dev/) + validação robusta
- Boas práticas de acessibilidade e performance

## Escopo do assessment

O desafio consistiu na criação de uma plataforma funcional que permitisse gerenciar e visualizar dados em diferentes contextos, simulando demandas reais da administração pública. Todo o projeto foi desenvolvido do zero: prototipação, UI/UX, consumo de APIs, tratamento de estados complexos e otimização de performance.
