Uma plataforma desenvolvida para o [Sebrae](https://sebrae.com.br/sites/PortalSebrae/). Foi desenvolvida com o intuito de centralizar os dados públicos de todas as empresas cadastradas no [Sebrae](https://sebrae.com.br/sites/PortalSebrae/), para que os usuários possam acessar e visualizar as informações de cada empresa.

Alem disso a plataforma oferece uma página de noticia e oportunidades de empresas, com informações sobre as empresas e as oportunidades que estão disponíveis. Essa plataforma foi apresentado no [Startup Summit 2023](https://agenciasebrae.com.br/inovacao-e-tecnologia/startup-summit-2023-chega-ao-fim-com-recorde-de-publico-e-numeros-que-impressionam/) e contou com mais de 10mil acessos simultâneos

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Todo o projeto teve um prazo super apertado, o que significa que a equipe de desenvolvimento trabalhou ativamente com a equipe de design para criar uma interface de usuário simples e intuitiva para que os usuários possam facilmente gerenciar suas plataformas e configurar acessos.

O desenvolvimento foi concluído dentro do prazo de 3 semanas, aonde foi feito o desenvolvimento da plataforma, configuração do servidor e deploy. Alem de outras configurações para a plataforma suportar mais de 10mil acessos simultâneos.

## Entrega do projeto

Por final, a plataforma foi entregue e apresentada no [Startup Summit 2023](https://agenciasebrae.com.br/inovacao-e-tecnologia/startup-summit-2023-chega-ao-fim-com-recorde-de-publico-e-numeros-que-impressionam/) e contou com mais de 10mil acessos simultâneos.
