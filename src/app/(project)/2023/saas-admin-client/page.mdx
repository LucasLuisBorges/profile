Uma plataforma saas desenvolvida para que administradores consigam criar e gerenciar seus usuários. Com uma interface de usuário simples e intuitiva, os administradores podem gerenciar suas plataformas, configurar os acessos, e gerenciar todos os dados exibidos na plataforma.

Esse projeto consiste em duas plataformas, uma para administração e outra para visualização de dados. [A plataforma de administração](https://www.admin.navint.com.br/) permite que os administradores criem usuários, configurem acessos e gerenciem todos os dados exibidos na plataforma. [A plataforma de visualização](https://www.navint.com.br/) permite que os usuários visualizem os dados exibidos na plataforma de administração.

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [React Query](https://react-query.tanstack.com/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Todo o processo de desenvolvimento foi feito de forma colaborativa, onde todos os membros do time de desenvolvimento trabalharam juntos para criar uma solução funcional e eficiente para o problema proposto. Desenvolvemos toda uma história de usuário, criando uma interface de usuário simples e intuitiva para que os usuários possam facilmente gerenciar suas plataformas e configurar acessos.

Utilizando o [draw.io](https://app.diagrams.net/), criamos uma diagrama de fluxo de dados para visualizar o processo de gerenciamento de usuários. A diagrama mostra como os usuários podem criar usuários, painéis, temas, configurar acessos e gerenciar todos os dados exibidos na plataforma. Além disso, utilizamos o [Figma](https://www.figma.com/) para criar uma interface de usuário visual e o [Tailwind CSS](https://tailwindcss.com/) para criar uma interface de usuário responsiva e personalizável.

Logo em seguida, antes do processo de desenvolvimento ser iniciado, fizemos a criação e separação de todas as regras de negócio, como as regras de negócio de gerenciamento de usuários, configuração de acessos, gerenciamento de dados e visualização de dados. Desenvolvemos essas regras de negócio para que os usuários possam gerenciar suas plataformas de forma eficiente e fácil.

## Entrega do projeto

Por final, depois do desenvolvimento de toda a plataforma de administração e visualização de dados, realizamos a entrega do projeto. Utilizamos o [Vercel](https://vercel.com/) para hospedar a plataforma e [AWS](https://aws.amazon.com/) para armazenar os dados no bucket e ficar responsável pelo nosso backend
