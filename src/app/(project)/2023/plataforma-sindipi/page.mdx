Desenvolvemos uma plataforma para o [Sindipi](https://sindipi.com.br/) com o objetivo de centralizar todas as suas plataformas em um único local. Dessa forma, os usuários podem acessar e visualizar as informações da empresa em tempo real.

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Toda a plataforma foi desenvolvida em um prazo de 3 semanas. Realizamos o desenvolvimento com integração a uma API fornecida pela equipe de dados.

O deploy foi feito na [Vercel](https://vercel.com/).

## Entrega do projeto

Por final, a plataforma foi entregue para o [Sindipi](https://sindipi.com.br/).
