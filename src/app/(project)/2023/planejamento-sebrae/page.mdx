Uma plataforma desenvolvida para o [Sebrae](https://planejamento.sebrae-sc.com.br/). Foi desenvolvida para complementar uma apresentação deles e concorrer a prémios.

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Todo o projeto teve um prazo super apertado, o que significa que a equipe de desenvolvimento trabalhou ativamente com a equipe de design para criar uma interface de usuário simples e intuitiva para que os usuários possam facilmente gerenciar suas plataformas e configurar acessos.

O desenvolvimento foi concluído dentro do prazo de 3 semanas, aonde foi feito o desenvolvimento da plataforma, configuração do servidor e deploy no [Bitbucket](https://bitbucket.org/). Alem de outras configurações para a plataforma suportar mais de 10mil acessos simultâneos.

## Entrega do projeto

Por final, a plataforma foi entregue e apresentada.
