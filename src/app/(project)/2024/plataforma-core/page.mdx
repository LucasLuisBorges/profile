Uma plataforma que reúne diversos painéis com informações relevantes e permite que os usuários visualizem e compartilhem essas informações.

Essa plataforma conta com alguns dados puxados diretamente de um [Google Sheets](https://docs.google.com/spreadsheets/u/0/?hl=pt) e todos os painéis vem de uma API interna da empresa.

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [React Query](https://react-query.tanstack.com/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Todo o processo de desenvolvimento foi feito em 4 semanas, contando com autenticação, painéis, visualização e compartilhamento de dados.

## Entrega do projeto

Por final, a plataforma foi subida na [Vercel](https://vercel.com/) e entregue para o [core](https://coresc.org.br/).
