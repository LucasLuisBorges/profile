Uma plataforma desenvolvida para a [Agencia Espacial Brasileira](https://www.aeb.org.br/) que reúne diversos painéis com informações relevantes e permite que os usuários visualizem e compartilhem essas informações.

Essa plataforma também conta com uma versão interna com dados privados, essa plataforma interna tem um layout bem inovador e animado. Mas não posso fazer a divulgação ou entrar em mais detalhes.

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [React Query](https://react-query.tanstack.com/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Todo o processo de desenvolvimento contou com criação do protótipo, aprovação do layout e assim foi iniciado o desenvolvimento da plataforma.

O processo de desenvolvimento levou uma semana para ser concluído.

## Entrega do projeto

Por final, a plataforma foi subida na [Vercel](https://vercel.com/) e entregue para a [Agencia Espacial Brasileira](https://www.aeb.org.br/).
