Uma plataforma que reúne diversos painéis com informações relevantes e permite que os usuários visualizem e compartilhem essas informações.

## Desenvolvido utilizando:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [React Query](https://react-query.tanstack.com/)
- [Shadcn/ui](https://ui.shadcn.com/)

## Processo de desenvolvimento

Plataforma desenvolvida para o [Sindipi](https://sindipi.com.br/). Foi desenvolvida com o intuito de centralizar apenas os painéis que são relevantes para o usuário.

Esse site é um espelho de uma das páginas da [Plataforma Sindipi](https://plataforma-sindipi.vercel.app/), com o intuito de deixar o usuário ter uma experiência mais simples e intuitiva.

## Entrega do projeto

Por final, a plataforma foi subida na [Vercel](https://vercel.com/) e entregue para o [Sindipi](https://sindipi.com.br/).
